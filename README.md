# NodeCollector

多协议节点URI采集与格式化工具，支持从公开数据源采集、本地库导入、格式转换导出。

## 免责声明

**本项目仅用于技术学习、网络协议研究与数据采集练习。**

- 本项目**不提供**任何代理服务、不运行任何代理服务器、不分发任何可用于绕过网络审查的配置
- 采集的URI来自公开互联网资源，项目本身**不包含、不存储、不传输**任何实际代理连接能力
- 用户须自行确保使用行为符合所在地区法律法规，**禁止用于任何违法活动**
- 项目不对采集内容的可用性、安全性、合法性做任何保证，使用后果自负
- 本项目**无用户群、无节点分享渠道**，仅提供本地数据采集与格式化功能
- 若您是数据源提供者并要求移除相关内容，请提交Issue，我们将立即处理
